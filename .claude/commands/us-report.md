米国株のリサーチレポートを、リサーチ・分析チームを使って作成してください。

## 手順
1. `US_EQUITY_PROFILE.md`(株式枠・中リスク統合)と `LESSONS.md` を読み、前提と教訓を踏まえる。
2. `us-equity-analyst` を起動し、本日(時点明記)の以下を Web 検索で収集・分析させる:
   - 地合い: S&P500/NASDAQ、米金利・FRB、USD/JPY、セクターの流れ
   - 個別: `$ARGUMENTS` に銘柄指定があればそれを対象に。なければ US_EQUITY_PROFILE.md の保有銘柄
   - 各銘柄でforward P/E or PSR・直近決算のビート/ミス・希薄化・為替/税の含意・リスクを必ず確認
3. `red-team-critic` に出力を渡して批判的検証(必須ゲート)。特に「バリュエーション過熱」「プレ収益銘柄の希薄化・ランウェイ」「為替リスク」「日本株/暗号資産とのAI・量子テーマ横断集中」を重点検証させる。
4. `report-editor` を起動し、レッドチーム指摘込みで統合 → `reports/us-YYYY-MM-DD.md` に保存、主要な見立てを `DECISION_LOG.md` に「[US]」タグ付きで追記させる。
5. 「今日の要点(3行)」と「最も注意すべき点」をチャットに要約する。

## 注意
- 米国株は円換算で為替影響を受ける旨、配当源泉税・申告分離の税制を明記する。
- 投機的・プレ収益銘柄(例: IONQ)は株式枠内でも「高リスク」と明示する。
- 株価は時点を明記し、未確認は断定しない。
