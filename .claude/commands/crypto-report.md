暗号資産(別枠「高リスク」)のリサーチレポートを、リサーチ・分析チームを使って作成してください。

## 手順
1. `CRYPTO_PROFILE.md` と `LESSONS.md` を読み、リスク方針(高リスク枠)と過去の教訓を踏まえる。
2. `crypto-analyst` を起動し、本日(時点明記)の以下を Web 検索で収集・分析させる:
   - 地合い: BTC価格・ドミナンス、ETH、Fear&Greed、ETFフロー、マクロ(米金利・ドル)、サイクル局面
   - 個別: `$ARGUMENTS` に銘柄指定があればそれを対象に。なければ CRYPTO_PROFILE.md の保有銘柄全て
   - 各銘柄でトークノミクス・アンロックの崖・規制・固有リスクを必ず確認
3. `red-team-critic` に出力を渡して批判的検証を行わせる(必須ゲート)。特に「保有銘柄の相関(アルトはBTC/リスク選好に連動して同時に動く)」「アンロックの崖」「新興L1・ミームの投機性」を重点検証させる。
4. `report-editor` を起動し、レッドチーム指摘込みで統合 → `reports/crypto-YYYY-MM-DD.md` に保存、主要な見立てを `DECISION_LOG.md` に「[CRYPTO]」タグ付きで追記させる。
5. 「今日の要点(3行)」と「最も注意すべき点」をチャットに要約する。

## 注意
- 暗号資産は高リスクで全額を失う可能性がある旨を、レポート冒頭に必ず明記する。
- 価格は24時間で大きく動くため、必ず取得時点を明記し、未確認は断定しない。
- 株式の「中リスク」基準を流用せず、高リスク枠として評価する。
