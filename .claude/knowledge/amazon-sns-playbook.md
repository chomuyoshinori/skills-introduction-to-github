# Amazon × SNS 外部集客 プレイブック（セミナー学習ノート）

広告費高騰への対策として、Instagram を中心に SNS から Amazon へ集客する手法と成功事例。
`sns-specialist` / `marketing-specialist` を中心に、チーム共通の前提知識とする。出典：社内セミナー資料。

---

## 1. Amazon広告の現状と課題（なぜ外部集客が必要か）

- **CPC上昇トレンド**：2026年には一部カテゴリーで **1クリック500円超**。
- **CPC変動幅**：プライムデー等では通常期の **1.5〜2倍** に上昇。
- **リスク構造**：広告費を下げると売上減、上げすぎると利益が消える **悪循環**。
  → Amazon 内広告（PPC）だけに依存しない、**SNSからの外部流入**が打開策。

---

## 2. SNS活用の意義と優位性

### Amazonアルゴリズムへの影響（最重要）
- **外部流入は Amazon の評価を高める**：内部検索からの流入が10点なら、**外部流入は12〜15点**の評価。
- CVR向上・自然検索順位の改善・商品ページ評価UPに寄与。
- 競合が少ないほど、早期参入で **一人勝ち** が可能。
- → 外部流入はアルゴリズム評価を **劇的に** 向上させる。

### 長期競合優位性
- 講師は2022年から SNS を本格運用 → **4年間で同ジャンルの競合不在を維持**。
- 継続的なコンテンツ提供で市場独占を実現。継続こそが堀（モート）になる。

## 3. Instagram の基本構造（役割分担）

| 機能 | 位置づけ | 役割 |
|---|---|---|
| **フィード投稿** | 定番 | 永久保存・検索可能。ブランド世界観・**信頼構築の基盤**。長期的認知。 |
| **ストーリーズ** | 即時 | 24時間で消える。**限定セール・新商品告知**に最適。リアルタイムの行動喚起。 |
| **リール** | 拡散 | ショート動画で**新規フォロワー獲得の鍵**。拡散力が非常に高い＝バズの発火点。 |
| **プロフィール** | 出口 | 唯一の恒久リンク設置場所。**Amazon商品ページ/ブランドページへの誘導の窓口**。ビジネスアカウント化でインサイト分析・連絡先ボタン。 |

> 役割：フィード＝信頼、ストーリーズ＝告知、リール＝新規拡散、プロフィール＝Amazonへの送客口。

## 4. コンテンツ設計の原則

### アルゴリズムに強いコンテンツ
- **保存数とシェア数が最も重視される**（いいね数より上位指標）。
- 「後で見返したい」要素（チェックリスト・比較表）を含める。
- 明示的なCTA：「**保存してね**」「**友達に教えてね**」で拡散率を上げる。

### リールの黄金構成
**フック → 共感 → 解決 → CTA**
- 冒頭に**インパクトフック**（「知らない?」「必見!」）で離脱を防ぐ。
- 閲覧維持 → 共感 → 解決 → CTA（クーポン案内）の流れ。
- バズりやすいテーマ：**ビフォーアフター / ライフハック / 意外な使い方 / 開封レビュー**。

## 5. 成功事例と共通要因

| ブランド | 成果 | ポイント |
|---|---|---|
| 美容スキンケア | 6ヶ月で Amazon ランキング **20位 → 2位** | 価値提供型コンテンツ（使い方・収納アイデア）、3ヶ月以上の継続投稿 |
| 雑貨 | **3ヶ月で検索順位が劇的改善** | 週4回の定期投稿＋リール運用、プロフィールとリンク誘導を明確に設計 |
| 食品 | リールがバズり短期間で小ヒット商品に成長 | 商品紹介より**価値提供型コンテンツを優先**、3ヶ月以上継続 |

**共通要因**：①価値提供型コンテンツ（売り込みより役立ち情報）②最低3ヶ月以上の継続 ③プロフィールからAmazonへの導線設計。

## 6. 支援体制と実践ヒント

### SNS代行サービス（外注の目安）
- **月額5万円から**のサービスがコストパフォーマンス最適。
- 高額な月額20〜30万円のサービスは質が不安定なこともあり、額面で選ばない。

### マイクロインフルエンサー提携
- **フォロワー1万〜3万人未満**のマイクロインフルエンサーに**ギフティング**（商品提供）。
- DMを地道に送り、リアルな口コミ（UGC）を生む。
- 投稿内容は**保存・シェアされやすい情報**であることが重要。
- → ギフティングでリアルな口コミを生むのがコツ。

> ⚠️ ギフティング・PR投稿は**ステマ規制（景表法）**の対象。「PR」「提供」等の明示が必須。
> 詳細は `legal-advisor` / `ethics-compliance` に確認すること。

---

## 7. 核心インサイト（総括）

- **SNS運用は「量」より「継続」が鍵**。
- **3ヶ月以上の継続**でアルゴリズムに認識され始める。
- **商品紹介より「価値提供型コンテンツ」が一人勝ち**を生む。
- 外部流入は Amazon アルゴリズム評価を高め、**広告費依存から脱却**する最大の武器。

---

## 8. チームの行動指針への落とし込み

- `sns-specialist`：フィード/ストーリーズ/リール/プロフィールの役割分担を前提に、3ヶ月継続・価値提供型・保存シェア設計でプランを組む。リールは「フック→共感→解決→CTA」。
- `marketing-specialist`：SNS外部流入を Amazon SEO/CVR 改善の施策として PPC とセットで設計する。
- `creative-designer`：保存したくなる比較表/チェックリスト型ビジュアル、開封・ビフォーアフターを意識した撮影。
- `data-analyst`：保存数・シェア数・プロフィール→Amazon送客（Attribution）を KPI に据える。
- ステマ規制・PR表示は必ず `legal-advisor` / `ethics-compliance` を通す。
