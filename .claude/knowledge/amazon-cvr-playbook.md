# Amazon CVR（転換率）向上 プレイブック（セミナー学習ノート）

商品画像・A+コンテンツ・ブランドストア（LP）・動画で **CVR を上げて売上を伸ばす**手法。
出典：社内セミナー資料「Amazon販売 CVR向上の実践講座」。`creative-designer` / `marketing-specialist` を
中心に、ページ改善・撮影ディレクション・ビジュアル戦略の前提知識とする。

---

## 0. CVRとは（なぜ最重要か）

- **計算式**：CVR ＝ 購入者数 ÷ 商品ページ訪問者数 × 100（%）。例：1,000人中50人購入＝5%。
- **Amazon平均は約10〜15%**（カテゴリにより差）。自社値はセラーセントラル
  「レポート → ビジネスレポート → ASIN別 → **ユニットセッション率**」で確認。
- **なぜ重要か**
  - 売上：同じアクセスでも売上が伸びる。
  - 検索順位：Amazonは **CVRの高い商品を上位表示**する。
  - 広告効率：CVRが上がると ROAS が改善し、広告費が活きる。

**CVRを上げる8施策の全体像**（本プレイブックは視覚系4つを深掘り）
商品画像の改善 / A+コンテンツ活用 / ブランドストア(LP)構築 / 動画挿入 ／（他：価格見直し・レビュー獲得・FBA活用・広告ターゲット最適化）。

## 1. 商品画像の最適化

**商品画像の3つの役割**
1. **規約遵守**（検索画面から除外されないため）
2. **CTR向上＝メイン画像**（競合の中からクリックされるため）
3. **CVR向上＝サブ画像**（ページを見た人を購入に導くため）

### メイン画像の絶対ルール（違反＝検索から除外）
| ルール | 内容 |
|---|---|
| 背景色 | **純白 RGB(255,255,255)**。グレー等はNG |
| 商品占有率 | 商品が画像全体の **85%以上** |
| テキスト | 装飾文字・キャッチコピー・スペック羅列は **NG** |
| ロゴ/透かし | 自社ロゴはOK、"Best Seller"等のバッジは **NG** |
| グラフィック | 矢印・吹き出し・アイコン挿入は **NG** |

- 推奨スペック：**1,600×1,600px以上**（ズーム機能のため）／JPG推奨／最大10MB。

### CTRを高めるメイン画像（検索画面でクリックされる3点）
1. 商品の最も魅力的な角度を演出（初見でも理解できるカット）。
2. 明るく鮮やかな撮影（自然光・影を抑え清潔感、**競合より明るく**）。
3. 抜けのある余白（サムネ縮小時の視認性を確保）。
> 無料ツール：Canva / Adobe Express / remove.bg。

### CVRを高めるサブ画像「8枚ストーリー設計」
購入を決断させる順番で並べる。
| 順番 | 役割 | 内容 |
|---|---|---|
| 1枚目(メイン) | 入口 | クリック誘導 |
| 2〜3枚目 | 機能 | ベネフィットを分かりやすく訴求 |
| 4〜6枚目 | 詳細 | スペック・使用シーンで不安を解消 |
| 7〜8枚目 | 信頼 | 比較表・レビュー・保証で安心させる |

> スマホ最適化必須：Amazonユーザーの約7割がスマホ。文字は大きく、**1枚1メッセージ**。

### 競合分析から「売れる画像」を設計（3ステップ）
- Step1：競合上位5〜10商品の**低評価レビュー（★1〜2）**を分析し不満点をリスト化。
- Step2：不満を画像で**事前解決**（「サイズが大きい」→サイズ比較画像、「説明書ない」→図解）。
- Step3：自社の独自性（USP）を画像で可視化。

**制作ツール**：Canva（サブ画像）/ Adobe Express（メイン）/ remove.bg（背景透過）/ Photoshop（高度加工）。
初心者は Canva 無料テンプレ → スマホで縮小確認 → 競合と並べて違いが一目で分かるか確認。

## 2. A+コンテンツの作成（商品ページをLP化）

- **A+とは**：商品ページ内に画像・テキスト・比較表を組み合わせて追加できる機能。LPのようなリッチページ。
  **ブランド登録済みの出品者のみ**利用可。導入でCVR向上の事例が多数。
- **4つの効果**：CVR向上（商品理解が深まる）／ブランドイメージ向上／信頼感アップ／競合差別化。
- **3種類**：① ベーシックA+（基本）② ブランドストーリー ③ **プレミアムA+**（動画・インタラクティブ要素、より高CVR）。
- **作成5ステップ**：① セラーセントラルにアクセス → ② 種類を選択 → ③ 構成とモジュールを配置（スマホで離脱しない**H2見出し7本構成**）→ ④ 対象ASINに適用 → ⑤ 審査提出（**承認まで通常7日程度**）。
- **おすすめモジュール3選**：テキスト付き標準画像ヘッダー（970×600px）／標準4画像＋テキスト（300×300px）／**商品の比較表**。
- **4つのポイント**：特徴を分かりやすく／高品質画像／モバイル意識／ターゲットに合わせる（初心者=使い方、上級者=スペック）。
- **禁止事項**：最上級表現「世界一」「最高」、**価格・割引・送料への言及**、Amazon外リンク・連絡先誘導、過剰な保証表現。

## 3. ブランドストア（LP）の構築

- **とは**：Amazon内に独自URLで作る**自社ブランド専用ページ**（ロゴ・画像・動画・ブランドストーリー）。ブランド登録者のみ。
- **4つのメリット**：① ブランディング強化（ファン・リピーター）② クロスセル/アップセル（複数商品を回遊）③ **広告効果最大化**（スポンサーブランド広告のリンク先にするとROAS改善）④ ストアインサイト（訪問者・流入分析）。
- **2つの必須条件**：① **大口出品プラン**（月4,900円）② **Amazonブランド登録**の完了。
- **作成6ステップ**：セラーセントラル「ストア」→ ロゴ（400×400px）→ テンプレ選択 → 商品・画像・動画追加 → プレビュー → 審査・公開。
- **スポンサーブランド広告との連携**：検索結果上部のバナー型広告（ロゴ・見出し・複数商品）。**リンク先をブランドストアにすると効果が最大化**。

## 4. 動画活用でCVRを底上げ

- **なぜ重要**：使い方・質感・サイズ感が伝わる／競合差別化／滞在時間UPでCVR向上／購入ギャップを減らし**返品低減**。
- **活用場所**：商品ページ／A+（プレミアムA+）／ブランドストア／スポンサーブランド広告。
- **商品ページへの挿入**：ブランド登録者はセラーセントラル「在庫 → 動画管理」からアップロード → ASIN指定 → 審査。
- **CVRを高める構成（1〜2分）**
  | 時間 | 内容 |
  |---|---|
  | 0〜5秒 | 商品名とインパクト（最初の5秒で惹きつける） |
  | 5〜20秒 | 課題提起と解決を見せる |
  | 20〜40秒 | 使い方・機能 |
  | 40〜50秒 | 利点・ベネフィット |
  | 50〜60秒 | 信頼・ブランドメッセージ |
- **ポイント**：**字幕必須**（音声オフ前提）／短く伝わる構成。無料の **Amazon動画ビルダー**でも制作可。

## 5. 実践ロードマップ & 効果測定

- **フェーズ1（今できる）**：商品画像の改善（メイン規約確認 → サブ8枚ストーリー → 競合分析）。
- **フェーズ2（1〜2週間）**：A+コンテンツの作成。
- **フェーズ3**：ブランドストア・動画の活用。
- **効果測定**：ビジネスレポートで**ユニットセッション率（CVR）**・セッション数・PVを確認。
  **PDCA**：Plan→Do→Check→Action。**変更後2〜4週間**でCVRの変化を見る。

## 6. よくある失敗と対処（10選・抜粋）

| 失敗 | 対処 |
|---|---|
| メイン画像の背景が純白でない | RGB(255,255,255)を必ず確認 |
| 楽天用の画像をそのまま流用 | Amazon専用に作り直す |
| サブ画像が角度違いだけ | ベネフィット・スペック・使用シーンを追加 |
| A+に価格情報を記載 | 価格・割引・送料の表現は禁止 |
| スマホで文字が読めない | 24pt以上・スマホで必ず確認 |
| 動画に字幕がない | 音声オフでも伝わる字幕を入れる |
| ブランドストアを放置 | 季節・キャンペーンで定期更新 |
| 競合分析せず画像を作る | 上位商品と比較してから制作 |
| 1枚に情報を詰め込みすぎ | 1画像1メッセージの原則 |
| 効果測定をしない | 変更後2〜4週間でCVR変化を確認 |

---

## 7. 総括：CVRを上げる4つの視覚施策

1. **商品画像の最適化**（規約遵守メイン＋8枚ストーリーのサブ、競合分析→USP可視化）
2. **A+コンテンツの活用**（ブランド登録済みなら無料、ページをLP化）
3. **ブランドストア(LP)の構築**（自社専用ページ＋スポンサーブランド広告連携でROAS最大化）
4. **動画の活用**（静止画で伝わらない「体験」を届ける）

> まず最初にやること：**自分の商品ページのCVR（ユニットセッション率）を確認する**。
> ※画像・A+・動画の表現は薬機法・景表法・Amazon規約の人手チェックを必須とし、`legal-advisor` / `ethics-compliance` を通す。
