# 書籍知識化プロファイル

書籍知識化チーム(`knowledge-*` エージェント)が作業前に必ず参照する設定。
**ここを更新すると、知識ノートの深さ・形式・トーンが変わる。**

## 読者像(誰のための学習ノートか)
- 本人(オーナー)の学習・理解整理用。初見の分野でも本質を掴めるレベルで噛み砕く。

## 用途
- [x] 学習・理解(概念を身につける)
- [x] 後から引けるリファレンス(用語集・チェックリスト)
- [ ] 人に教えるための教材化
- [ ] 実務への即適用(手順・テンプレート重視)
※ 必要に応じてチェックを更新する。

## 出力の好み
- 言語: 日本語
- 深さ: 中〜深(要点だけでなく、根拠・限界・反証まで)
- 図解: 積極的に使う(Mermaid・対比表)
- 長さ: 1冊あたり読み切れる分量に圧縮(冗長さより本質)
- トーン: わかりやすく、しかし正確。過剰単純化を避ける。

## 必ず守るルール
1. 書籍本文の丸写し・大量転記をしない(要約・再構成のみ。引用は短く・出典明記)。
2. 市販書籍のPDF本体はリポジトリにコミットしない(`books/` は `.gitignore` 済み)。
3. Webで補った事実は出典・時点を残す。未確認は「未確認」と書く。
4. 著者の主張と、確認された事実を区別して書く。
5. 批判的検証(`knowledge-critic`)と倫理チェック(`knowledge-ethicist`)を必ず通す。

## 保存先
- 完成した知識ノート: `knowledge/<書名>.md`
- 元のPDF/原稿(任意): `books/`(Git管理外)
